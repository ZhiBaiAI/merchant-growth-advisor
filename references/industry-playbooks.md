# Industry Playbooks

Use these as starting points. Adapt to the user's city, customer profile, average order value, season, and existing channels.

## 餐饮

- Focus: meal periods, set meals, group-buying packages, neighborhood traffic, office workers, family gatherings.
- Good campaigns: weekday lunch card, limited tasting set, old customer bring-a-friend, birthday table gift, late-night combo.
- Useful copy angles: "今天不用想吃什么", "附近上班族午餐", "下班后来一份热乎的".
- Channels: 抖音团购、美团/大众点评、微信社群、朋友圈、外卖平台店铺页.

## 美业

- Focus: trial experience, before/after cases, appointment filling, membership package, referral.
- Good campaigns: first-order experience, old customer referral card, seasonal skin/hair/nail theme, idle-time discount.
- Useful copy angles: "周末前变精致", "通勤也能做的轻护理", "先体验再办卡".
- Channels: 小红书、视频号、朋友圈、社群、门店顾客转介绍.

## 健身

- Focus: low-pressure trial, body assessment, class attendance, habit formation, private coaching conversion.
- Good campaigns: 7-day check-in, body test appointment, group class open day, partner training card.
- Useful copy angles: "不用硬扛，先建立习惯", "下班后一小时", "带朋友一起动起来".
- Channels: 短视频、本地社群、企业团体、朋友圈案例.

## 教培

- Focus: trial class, parent trust, learning outcome, schedule fit, referral.
- Good campaigns: free assessment, trial class week, parent open class, old student referral reward.
- Useful copy angles: "先诊断再报名", "看得见的学习反馈", "适合附近家庭的时间安排".
- Channels: 社群、朋友圈、视频号、地推、异业合作.

## 零售

- Focus: footfall, new arrivals, bundles, membership points, repeat visit.
- Good campaigns: new-arrival day, limited bundle, member day, clearance with story, cross-sell set.
- Useful copy angles: "今天到店有新款", "顺路带走", "会员专享".
- Channels: 朋友圈、社群、短视频、门店橱窗、会员短信.

## 母婴

- Focus: trust, safety, neighborhood relationships, parent education, repeat purchase.
- Good campaigns: parenting micro-class, member replenishment reminder, baby milestone gift, neighborhood parent group.
- Useful copy angles: "给新手爸妈少踩坑", "附近妈妈都在问", "按月龄选更省心".
- Channels: 社群、朋友圈、线下活动、异业合作.

## 宠物

- Focus: pet owner emotion, service cases, grooming/boarding trust, repeat reminders.
- Good campaigns: first grooming trial, birthday photo, vaccination/grooming reminder, pet social day.
- Useful copy angles: "让毛孩子舒服一点", "第一次到店也安心", "记录它今天的可爱".
- Channels: 小红书、短视频、朋友圈、社群、异业合作.
